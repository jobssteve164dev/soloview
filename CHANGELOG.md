# Changelog

## Unreleased

- 重写中英文市场介绍，突出本地优先、只读安全与减少应用切换的核心价值。
- 增加正式市场 Logo。

## 0.1.1

- 增加 SoloView 活动栏图标和“最近打开”侧边栏。
- 最近文档按打开时间排序，点击即可重新打开，并支持清空本地记录。

## 0.1.0

- 首次发布 SoloView。
- 支持在 VS Code 标签页中本地查看 PDF、DOCX、XLSX、XLS、CSV 与 PPTX。
- 提供统一缩放、重新载入、工作表切换和错误恢复体验。
- 默认只读，不执行 Office 宏，不上传文档。
